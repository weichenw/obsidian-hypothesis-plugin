Template (Nunjucks) for rendering every synced Hypothesis highlights & annotations.

Available variables to use

Article Metadata
  • {{is_new_article}} - New file indicator
  • {{title}} - Title
  • {{author}} - Author
  • {{url}} - Link to source
  • {{highlights}} - List of your Highlights
Highlight
  • {{id}} - Unique Id
    May contain the character _ ,
    cannot be used as an obsidian block reference id
  • {{id_base62}} - Unique id after base62 encoding
    Character sets:
    0 to 9, A to Z, a to z
  • {{text}} - Text
  • {{color}} - Highlight color
  • {{incontext}} - Link to Highlight in context
  • {{created}} - Created on
  • {{updated}} - Updated on
  • {{user}} - Username
  • {{group}} - Group name
Annotation
  • {{annotation}} - Annotation
  • {{tags}} - List of tags
